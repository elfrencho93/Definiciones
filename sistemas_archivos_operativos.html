Tipos de Sistemas de Archivos y Sistemas Operativos


¿Que es un Sistema de Archivos?

Un sistema de archivos es la forma en que se nombran los archivos y dónde se colocan lógicamente para su almacenamiento y recuperación. Sin un sistema de archivos, la información almacenada no estaría aislada en archivos individuales y sería difícil de identificar y recuperar. A medida que aumentan las capacidades de datos, la organización y la accesibilidad de los archivos individuales se vuelven aún más importantes en el almacenamiento de datos.


Los sistemas de archivos digitales y los archivos tienen el nombre y el modelo de los sistemas de archivo en papel que utilizan el mismo método basado en la lógica para almacenar y recuperar documentos.


Los sistemas de archivos pueden diferir entre los sistemas operativos (SO), como Microsoft Windows, macOS y sistemas basados en Linux. Algunos sistemas de archivos están diseñados para aplicaciones específicas. Los principales tipos de sistemas de archivos incluyen sistemas de archivos distribuidos, sistemas de archivos basados en disco y sistemas de archivos de propósito especial.


Tipos de Sistemas de Archivos

Hay varios tipos de sistemas de archivos, todos con diferentes estructuras lógicas y propiedades, como velocidad y tamaño. El tipo de sistema de archivos puede diferir según el sistema operativo y las necesidades de ese sistema operativo. Los tres sistemas operativos de PC más comunes son Windows, Mac OS y Linux. Los sistemas operativos móviles incluyen Apple iOS y Google Android.


Sistema de Archivo Definición Sistema Operativo
FAT Se considera simple y confiable, y se basa en los sistemas de archivos heredados. FAT se diseñó en 1977 para disquetes, pero luego se adaptó para discos duros. Si bien es eficiente y compatible con la mayoría de los sistemas operativos actuales, FAT no puede igualar el rendimiento y la escalabilidad de los sistemas de archivos más modernos. Windows
GFS Es un sistema de archivos de disco compartido. GFS ofrece acceso directo al almacenamiento de bloques compartidos y se puede utilizar como un sistema de archivos local.

GFS2 es una versión actualizada con características no incluidas en el GFS original, como un sistema de metadatos actualizado. Según los términos de la Licencia pública general de GNU, los sistemas de archivos GFS y GFS2 están disponibles como software gratuito.		 Linux
HFS HFS también puede denominarse Mac OS Standard, y fue reemplazado por Mac OS Extended. Introducido originalmente en 1985 para disquetes y discos duros, HFS reemplazó el sistema de archivos original de Macintosh. También se puede utilizar en CD-ROM. Mac OS
NTFS Es el sistema de archivos predeterminado para los productos de Windows a partir del sistema operativo Windows NT 3.1. Las mejoras del sistema de archivos FAT anterior incluyen un mejor soporte de metadatos, rendimiento y uso del espacio en disco. NTFS también es compatible con el sistema operativo Linux a través de un controlador NTFS gratuito y de código abierto. Los sistemas operativos Mac tienen soporte de solo lectura para NTFS. Windows, Linux y Mac OS
UDF Es un sistema de archivos independiente del proveedor que se utiliza en medios ópticos y DVD. UDF reemplaza el sistema de archivos ISO 9660 y es el sistema de archivos oficial para video y audio DVD elegido por DVD Forum. Windows, Linux y Mac OS
